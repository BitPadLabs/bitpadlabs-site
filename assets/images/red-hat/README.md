# Red Hat Brand Assets

This directory contains approved Red Hat brand assets for use in BitPadLabs partnership materials.

## Required Assets

### Logos
- `RedHat-Logo-Corporate.svg` - Primary Red Hat corporate logo (full color)
- `RedHat-Logo-Corporate.png` - PNG version for web use
- `RedHat-Logo-White.svg` - White version for dark backgrounds
- `RedHat-Logo-Black.svg` - Black version for single-color applications

### Co-Branding Template
- `co-branding-template.svg` - Official Red Hat co-branding template for creating lockups

## How to Obtain Assets

Red Hat brand assets are available through:
1. **Red Hat Partner Connect** - Login required for partner access
2. **Red Hat Partner Team** - Contact for specific asset requests
3. **Official Template:** https://www.redhat.com/rhdc/managed-files/rhb-logos-co_branding-template_0.svg

## Usage Guidelines

All Red Hat brand assets must be used in accordance with:
- Red Hat Partner Handbook: https://www.redhat.com/en/about/brand/standards/partner
- BitPadLabs Copilot Instructions: .github/copilot-instructions.md

### Key Rules:
- Always use Red Hat corporate logo (not product logos)
- Never use Red Hat logo in grayscale
- BitPadLabs logo comes first in partner-led co-brands
- Use full Red Hat product names (never abbreviate)
- Maintain adequate clear space around logos

## Notes

Logo files should be obtained directly from Red Hat through official partner channels to ensure you have the most current versions and proper licensing.
