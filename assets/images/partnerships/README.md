MIRO LOGO PLACEHOLDER

To use the Miro logo on the partnerships page:

1. Download the official Miro logo from their brand assets or partner portal
2. Save it to this directory as 'miro-logo.png' or 'miro-logo.svg'
3. Update partnerships.html to reference the correct filename

Recommended formats:
- SVG for scalability
- PNG with transparent background
- Minimum 200px width for web display

Usage guidelines:
- Maintain proper spacing around the logo
- Don't modify or distort the logo
- Use approved color variations only
